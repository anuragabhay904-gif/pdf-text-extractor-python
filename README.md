# PDF Text Extractor (Python)

## Overview
This project is a simple Python-based utility to extract text from PDF files using the PyPDF2 library.  
It is designed as a foundational project to understand file handling, third-party libraries, and text processing in Python.

## Features
- Extracts text from PDF files page by page
- Handles multiple pages automatically
- Lightweight and easy to extend
- Suitable for beginners learning Python and document processing

## Technologies Used
- Python 3
- PyPDF2

## How It Works
1. The program loads a PDF file from the local system.
2. Each page of the PDF is read sequentially.
3. Text content is extracted and printed or processed as required.
